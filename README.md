## Description

<!-- Plugin description -->
This plugin uses AI to automatically generate commit messages based on the changes in your code.

### Features

- Generate commit messages from staged changes.
- Support `OpenAI`, `OpenAI-Response`, `Gemini`, and `Anthropic`.
- Support refreshing model lists from provider APIs and manually entering custom model names.
- Support project-level prompt overrides via `commit-prompt.txt`.
<!-- Plugin description end -->

## Features

- Support configuring multiple 'prompts'.
- Support project-level personalized configuration.
- Use the 'prompt' from the 'commit-prompt.txt' file located in the project’s root directory.

## Supported Protocols

- [OpenAI](https://platform.openai.com/docs/overview)
- [OpenAI Responses](https://platform.openai.com/docs/api-reference/responses)
- [Gemini](https://ai.google.dev/gemini-api/docs)
- [Anthropic](https://docs.anthropic.com/en/api/overview)
